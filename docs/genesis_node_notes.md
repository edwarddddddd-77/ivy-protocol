# Ivy Protocol 创世节点先锋卡 (Genesis Node) 阅读笔记

## 1. 产品定义 (Product Definition)

**名称:** Ivy Genesis Pioneer Pass (常青藤创世先锋卡)

**定位:** 生态超级节点、无限代收益捕获器、治理权凭证

**发行总量:** 1,000 - 2,000 张 (建议稀缺性)

**视觉概念:** 赛博格风格的"能量核心"或"机械心脏"

## 2. 核心权益体系 (Core Utility Matrix)

### 2.1 权益一: 原生算力双向飞升 (Dual-Side Boost)

**自身增幅 (Self Boost):**
- 持有者在协议内进行 PID 挖矿或 Bond NFT 质押时，自身算力权重永久 **+10%**

**团队光环 (Team Aura):**
- 创世节点自带"祝福光环"
- 凡是填写创世节点邀请码的用户，其挖矿算力自动 **+2%**
- 营销话术: "填写我的邀请码，你挖得比别人快。"

### 2.2 权益二: 无限代级差分销 (Infinity Differential Rebate)

**采用抗泡沫的级差阻断模型:**

| 层级 | 普通用户奖励 | 创世节点 (NFT) 奖励 | 说明 |
|------|------------|------------------|------|
| L1 直推 | 10% | 10% | 获取直接邀请用户产出的 10% |
| L2 间推 | 5% | 5% | 获取直推用户再邀请人的产出 5% |
| L3~∞ 无限代 | 0% | +2% (级差统筹) | 节点独享。拿伞下无限层级业绩的 2%，直到遇到下一个节点 |

**级差阻断机制 (Breakaway):**
- 当您伞下的某条线出现新的"创世节点"时，该节点以下的 2% 统筹奖归该新节点所有
- 阻断发生后，原上级节点不再获得该条线的无限代奖励

**平级管理奖 (Management Bonus):**
- 为鼓励节点培养新节点，当阻断发生时，上级节点可获得该下级节点团队业绩的 **0.5%** 作为管理津贴

### 2.3 权益三: 生态金铲子 (The Golden Shovel)

**免税特权:**
- 豁免协议交互中的动态交易税 (通常为 0.2%)，实现无损进出

**空投权重:**
- 在 Ivy Protocol 未来的代币空投 (TGE) 中，持有 NFT 的地址享有最高权重系数

**Launchpad 优先权:**
- 未来生态孵化的子项目，创世节点自动获得白名单 (WL)

## 3. 市场推广整体方案 (Go-to-Market Strategy)

基于"节点级差"模型，推广重心不是"卖图"，而是**"招募合伙人"**。

### 3.1 目标客群 (Target Audience)

| 客群 | 关注点 |
|------|--------|
| 带单 KOL / 社区长 | 10% 直推 + 5% 间推 + 2% 无限代的恐怖裂变能力 |
| DeFi 大户 (Whales) | 10% 的自身算力加成和免税权 |
| 羊毛工作室 | "团队光环"，购买一张节点卡作为主号，然后下面挂几百个子号吃 +2% 加成 |

### 3.2 推广阶段规划 (Phases)

**阶段一: 猎头期 (Headhunting) - [T-14 ~ T-7]**
- 动作: 定向私聊 50-100 位币圈活跃的团长/KOL
- 话术: "我们不是在卖 NFT，我们在招募生态合伙人。这个机制设计了无限代级差，只有前 1000 人能拿这个 2% 的管道收益。"
- 激励: 给予早期承诺推广的 KOL 白名单 (WL) 打折铸造权

**阶段二: 裂变期 (Viral Launch) - [T-0 Mint Day]**
- 核心事件: 开启 Mint
- FOMO 机制: 配合"团队光环"进行宣传
- 宣传语: "没有邀请码只能裸挖！全网寻找创世节点邀请码，立享 2% 加速！"
- 效果: 散户在推特/电报群疯狂求邀请码，倒逼 KOL 必须去 Mint 一张卡来发码

**阶段三: 阵地战 (Territory Expansion) - [Post-Mint]**
- 数据看板: 官网上线"节点排行榜"，展示谁的伞下算力最高，谁拿的级差奖最多
- 造富效应: 官方定期以此宣传——"恭喜节点 A，本周仅靠级差奖躺赚 5,000 U。"

## 4. 经济模型风控 (Sustainability Check)

**非庞氏来源:**
- 所有返佣奖励 (10%+5%+2%) 均来自"代币经济学-挖矿产出池 (70%)"中的营销预算，或者是"用户挖矿产出的手续费抽成"
- 绝不挪用用户本金

**通胀控制:**
- 级差奖是同级阻断的，因此系统支出的最大总成本被锁定为 **17.5% (10+5+2+0.5)**
- 无论层级多深，泡沫都是可计算、可控的

## 5. 给设计团队的执行建议 (Design Brief)

**NFT 卡面:**
- 需体现"层级感"
- 建议设计 3 种不同材质 (银、金、黑金)，对应盲盒开出的不同稀有度
- 虽然权益一样，但稀有度可影响未来的空投数量

**Dashboard (仪表盘) UI:**
- 节点持有者登录官网后，必须看到一个"分销控制台"
- 显示内容:
  - 直推人数
  - 团队总人数 (无限代)
  - 累计获得的级差奖励
  - 下级节点阻断提醒

**体验感:**
- 要让他感觉自己不是在玩币，而是在管理一家公司
